## Introduction

This is the repository for the website running at https://spiritvale.info.

## Tech Stack
The backend is written in PHP (Tempest Framework) and the frontend is using Vue.js.
The connection is done using InertiaJs.

## Getting Started
Copy the .env.example file to .env and make optional changes.
Run `composer install` and `npm install` to install dependencies.
Run `php tempest serve` and `npm run dev` to start backend and frontend.
The site will be available at http://localhost:8000.
All data required to run the site will be loaded from the example-data directory.

## Example Data
The example data directory is supposed to provide the data required to run the site for testing and development.
It will not be updated to include new data after patches.
The example data will only be updated of the structure of the data changes.

## API
There is currently no API to fetch the game data.
An API will be added in the future.
